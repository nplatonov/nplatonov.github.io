Обработка

Есть атрибут (поле) “время”, определяющий последовательность локаций

Фунционал библиотеки adehabitatLT

  • Преобразования “sf ⬌ ltraj” вручную или через wildlifeDI::ltraj2sf(), wildlifeDI::sf2ltraj()
wildlifeDI::sf2ltraj(loc,id="id",date="time")
  • Выделение подтраекторий по пропускам во времени
adehabitatLT::cutltraj(x,"dt>c(days=1)*c(hour=1)*c(min=12)*c(sec=60)",nextr=TRUE)
  • Сегментация по устойчивому изменению параметров
res1 <- adehabitatLT::lavielle(x,Lmin=Lmin,Kmax=Kmax,type="mean")
nseg <- adehabitatLT::chooseseg(res1,output="opt",draw=TRUE)
res2 <- adehabitatLT::findpath(res1,K=nseg,plotit=TRUE)

Задание для знакомых с ГИС:

Дано:

  • Исходные данные: “(data/2022-01-09_14-00_Sun.gpx)[data/2022-01-09_14-00_Sun.gpx]” (“Лыжная прогулка”)

  • Начало и окончание на лыжной базе. Лыжник сделал два круга.

Определить:

  • Точку начала круга

  • Длину круга

  • Скорость прохождения каждого круга

  • Максимальное удалеление от бызы

Визуализация

da <- as.data.frame(sf::st_coordinates(loc))
da$time <- loc$time
da$z <- loc$ele
da$color <- as.integer(as.factor(loc$time))
da$color <- da$color/max(da$color)
da$text <- as.character(da$time)
s3d <- plotly::plot_ly(da,x=~X,y=~Y,z=~z,type='scatter3d',mode='lines'
              ,opacity=1
              ,hoverinfo=~text
              ,line=list(width=6
                        ,color=~color
                        ,colorscale=list(c(0,'#BA52ED'),c(1,'#FCB040'))
                        ,reverscale=FALSE
                        )
              )
s3d
sf::st_agr(tr) <- "constant"
center <- sf::st_coordinates(sf::st_centroid(tr))[1,]
dm <- da |>
   plot_ly(type="densitymapbox"
          ,lat=~Y
          ,lon=~X
          ,coloraxis="coloraxis"
          ,radius=10) |>
   layout(mapbox=list(style="stamen-watercolor"
                     ,center=list(lon=center["X"],lat=center["Y"])
                     ,zoom=15)
         ,coloraxis=list(colorscale = "Viridis"))
