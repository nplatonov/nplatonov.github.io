Биотелеметрия

Локации белого медведя DOI: 10.5066/P9ZRJ3XU.

Должны быть сформированы следующие файлы:

  • Местоположения отдельных особей для построения участка обитания (геометрия POINT)

  • Местоположения отдельных особей для построения траекторий (геометрия POINT)

  • Выделенные контуры кернёл по уровням 50% и 95% (геометрия POLYGON)

GPS

Запись траектории с лыжной прогулки, выполненная приложением OsmAnd в формате GPX
