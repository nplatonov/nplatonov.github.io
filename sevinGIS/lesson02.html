Установка

Standalone

https://qgis.org/en/site/

Находим “Download Now”

https://qgis.org/en/site/forusers/download.html

OSGeo4W

Программное обеспечение с открытым исходным кодом для Windows https://trac.osgeo.org/osgeo4w/

Находим “Download the ​OSGeo4W network installer”

NextGIS QGIS

https://nextgis.com/nextgis-qgis/ (Windows)

Находим “Download NextGIS QGIS installer”
