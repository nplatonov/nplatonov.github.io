R

Общие сведения

Программное обеспечение для обработки, анализа и визуализации данных

Легко устанавливается (один файл-установщик) и легко расширяется (одна строка)

Последовательность действий записывается в скрипт; интерпретатор (тоже R) его выполняет

Скрипт на языке (тоже R) высокого уровня, ориентированный на векторизацию

У R есть среда

  • терминал

  • графическая оболочка

  • RStudio

У R есть документация, примеры данных и объясняющие «виньетки».

У R есть комьюнити

Поисковые системы понимают запросы по R (“in R how to…”)

ChatGPT умеет кодить на R

Структура

R – модульная системы, расширяется за счет библиотек, упакованных в пакеты.

Если задача востребованная, есть вероятность, что она решена с помощью R.

У R есть основа – core:

  • base – базовые функции

  • datasets – наборы данных, на которых можно обыграть возможности R

  • utils – утилиты, ввод/вывод

  • graphics – графики

  • stats – основной статистический функционал

  • methods – методы для манипуляций с объектами более строгой типизации S4.

Расширения – contributed

  • их много, почти 20 тыс в CRAN

  • еще есть github (некоторые из-за несоответствия политике CRAN)

  • BioConductor

Rmarkdown

Markdown – язык разметки для обозначения форматирования в простом тексте.

Rmarkdown – включение фрагментов кода R в markdown

Код R выполняется в Rmarkdown (“knit”), и временно создаются чистый markdown и необходимые для подключения компоненты, например, рисунки.

Утилита “pandoc” преобразует markdown в требуемый формат: HTML, DOCX, PDF, PPTX

Вместе “knitr” + “pandoc” – это “render”
