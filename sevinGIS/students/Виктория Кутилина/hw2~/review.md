+ Сетка широт/долгот должны быть поверх Австралии

+ Загружать неиспользуемые библиотеки нет необходимости

+ Для отображения `oceans` не нужно использовать яркий цвет (`color = "darkblue"`) для границ; обычно границы морей не отображаются

+ Попробовать не переводить название морей вручную, а использовать столбец `name_ru` из таблицы атрибутов
