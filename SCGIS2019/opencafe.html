Abstract


Инструменты ГИС в первую очередь предназначены для разработчика проекта. Конечного пользователя часто интересуют обобщенные результаты без деталей разработки. Наглядное представление информации может быть достигнуто с помощью R и Shiny.
R содержит инструменты для анализа и визуализации, в том числе пространственных, данных и создания интерактивного веб-приложения без углубления в детали веб-разработки – shiny.
Демонстрация получения отчетной информации по пространственным запросам на примере проекта WWF по картированию экологически репрезентативной пан-арктической сети морских районов для управления биоресурсами и сохранения биоразнообразия.

«Marxan» - инструмент для стохастического поиска (эмуляция отжига) лучшей конфигурации территориальных единиц

Минимизация целевой функции:

\[\sum_{PUs} Cost + BLM \sum_{PUs} Boundary + \sum_{ConValue} SPF \times Penalty + CostThesholdPenalty\]

  1. \(\sum_{PUs} Cost\) - стоимость площади элемента

  2. \(BLM \sum_{PUs} Boundary\) - стоимость протяженности границ

  3. \(\sum_{ConValue} SPF \times Penalty\) - штраф за недостижение целевого показателя

  4. \(CostThesholdPenalty\) - штраф за перевыполнение целевого показателя (не использовался)
